# 🎥 YouTube RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot that allows users to ask questions about any YouTube video using its transcript.

---

## 🚀 Features
- Paste any YouTube video link
- Extracts transcript automatically
- Splits and embeds transcript using FAISS
- Semantic search over video content
- ChatGPT-style UI using Streamlit
- Timestamp-aware answers
- Chat memory support

---

## 🧠 Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace Embeddings
- Mistral AI
- YouTube Transcript API
